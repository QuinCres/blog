Github搭建个人博客
2019.8.20 星期二 15:32 #基础知识
第一步：登录GitHub，创建新仓库，并在设置中选择 一种主题。
第二步：打开Git Bash，通过cmd命令进入相应路径， 输入git clone + https的url。之后会在本地生成一个文件夹，删除文件夹中 除git外所有文件，并添加自己已写好的网页文件。
第三步：在Git Bash中输入“git add *”把当前要提交的代码上传 到缓存中,再输入“git commit -m "del"将缓存区内容添加到本地仓库” ,最后输入"git push origin master"把本地仓库文件上传到远程仓库
最后，刷新一下你的网页，就成功开始你的博客之路了。（提交后可能会有延迟， 多刷新几次就好了。）
